# DDD Note Type

Single note type: `Brazil DDD`.

One note per `DDD` code.

## Card templates

1. `DDD + blank municipal map -> locator`
2. `locator -> DDD`

## Core fields

- `ddd_code`
- `states`
- `municipality_count`
- `missing_geometry_municipality_count`
- `sample_municipalities`
- `blank_map`
- `locator_map`

## Design goals

- keep the map dominant
- keep the visual theme understated and map-first
